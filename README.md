# 📈 Stock Price Prediction using Machine Learning with Hyperparameter Tuning

## 📌 Problem Statement
The objective of this project is to predict future stock prices using historical market data.

Stock price prediction helps investors make better financial decisions by analyzing past trends and patterns.

---

## 📊 Dataset Information
The dataset contains historical stock market data including:

- Open Price
- High Price
- Low Price
- Close Price
- Volume
- Date

Target Variable:
- Next Day Closing Price (Regression Problem)

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Train-Test Split
5. Feature Scaling
6. Model Building
7. Hyperparameter Tuning using GridSearchCV / RandomizedSearchCV
8. Model Evaluation

## 🤖 Models Used
